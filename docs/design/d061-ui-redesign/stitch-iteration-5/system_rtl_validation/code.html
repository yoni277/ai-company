Logical CSS Strategy

swap_horiz

Directional Invariants

We utilize padding-inline-start and margin-inline-end to automatically adjust spacing based on the document's dir attribute. No duplicated CSS required.

auto_timer

Icon Mirroring

Directional icons (arrows, gavels, back buttons) are tagged with .rtl-mirror. In RTL mode, these automatically flip via transform to maintain semantic intent.

analytics

Visual Weight Balance

Text alignment (text-start) ensures content flows naturally from the edge, while fixed-width elements like the SideNav shift axes to preserve the "Anchor" feel.
